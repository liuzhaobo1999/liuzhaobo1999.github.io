Hugo使用指南

六月 6, 2020 发布在 hugo, theme

开始

先安装go和git

  1. win10 直接去hugo github下载EXE, 配置好环境变量. (Mac: brew install hugo)
  2. hugo new site /path/to/site
  3. cd /path/to/site
  4. hugo new about.md
  5. hugo new post/first.md
  6. mkdir themes && git clone https://github.com/kakawait/hugo-tranquilpeak-theme.git themes/tranquilpeak, 进入主题目录找到示例的配置文件, 替换config.toml
  7. hugo 生成静态文件public
  8. hugo server http://localhost:1313查看效果
  9. 把public部署到GitHub: 创建仓库yourname.github.io, 然后像git以往的操作, push到该仓库, 最后直接访问https://yourname.github.io
  10. 开发测试-快捷操作: ./dev.sh
#!/bin/bash
hugo

hugo server
  1. 部署-快捷操作: ./deploy.sh
#!/bin/bash
# build the project
hugo

cd public

git add .

msg="rebuilding site `date`"

if [ $# -eq 1 ]
  then msg="$1"
fi

git commit -m "$msg"

# push source to github

git push origin master

# come back to blog root

cd ..
